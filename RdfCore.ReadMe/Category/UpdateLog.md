## 基于Apollo实现统一配置中心
- 携程Apollo配置中心在.NET Core项目快速集成；
- .NET Core微服务之基于Apollo实现统一配置中心；
> Apollo（阿波罗）是携程框架部门研发的分布式配置中心，能够集中化管理应用不同环境、不同集群的配置，配置修改后能够实时推送到应用端，并且具备规范的权限、流程治理等特性，适用于微服务配置管理场景。
- Spring Cloud Config；
- Apollo集成了Eureka注册中心；
- Apollo对Java（1.8+）和MySQL（5.6.5+）的版本有要求；
- Apollo服务端基于Spring Boot和Spring Cloud开发（对的，就是微服务框架）；
